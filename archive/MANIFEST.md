# ARCHIVE MANIFEST: Primary Research Backup to AWS Glacier

This defines what gets backed up to Glacier and the rule for deciding. The pipeline downloads sources into `data/raw/`; the archive script (`archive/glacier_archive.sh`, generated by the build) uploads the items below. **The user runs the upload locally** against their saved AWS profile and vault. Credentials are never stored in this repo and never entered by the build process.

## The rule

Archive the **complete file** wherever the pipeline downloaded the full document or dataset. Where only a partial, abstract, or paywalled source was reachable, archive a **citation pointer** (a small JSON/text stub with title, author, URL, vintage, and access note), not a scraped fragment. A half-saved report is worse than a clean pointer to it.

Scope is **primary research only**: raw datasets and full report PDFs. Cleaned/derived data, JSON outputs, and all code stay in GitHub and are NOT sent to Glacier.

## Manual-fetch items (flagged for the user)

These cannot be fetched by an automated process because they sit behind a license click or paywall. Download once, drop into `data/raw/manual/`, and the archive script will include them:
1. EIG DCI full ZIP/county dataset (license click with institutional email).
2. Any paywalled journal PDFs you want preserved in full.

## What the archive script does

`archive/glacier_archive.sh` (generated by the build):
1. Reads `data/raw/` (auto-downloaded Tier 1 files) and `data/raw/manual/` (user-supplied).
2. Builds a single timestamped archive (tar) plus a manifest checksum file.
3. Uploads to the user's saved Glacier vault using the user's saved AWS profile.
4. Writes `archive/last_backup.json` with the archive ID, checksum, file list, and timestamp.

The script reads the AWS profile name and vault name from environment variables or the user's existing AWS config; it does not contain secrets. The user reviews the file list, then runs the upload. The upload is an irreversible write the user performs; the build never executes it.
