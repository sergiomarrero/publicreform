# Manual-fetch drop folder

Some primary sources cannot be fetched by an automated process because they sit
behind a license click or a paywall. Download them once, drop the files here,
and `archive/glacier_archive.sh` will include them in the next backup.

Flagged manual-fetch items (see `archive/MANIFEST.md`):

1. EIG DCI full ZIP/county dataset (license click with an institutional email).
2. Any paywalled journal PDFs you want preserved in full.

Files placed here are treated as Tier 1 (full files) for the Glacier archive.
Cleaned or derived data does not belong here; only primary research files.
